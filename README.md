# NomosIQ Marketing Site

React, Vite, TypeScript, Tailwind CSS, and shadcn/ui marketing site for NomosIQ.

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```

## Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`
- SPA fallback: `public/_redirects`
